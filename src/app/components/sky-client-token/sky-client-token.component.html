Sky Client
Sky Client is an Electron application, using the PepperFlash plugin in order to allow the continuation flash of usage.
Download on Windows

Please note: Windows may alert you that the application isn't safe - simply press More Info > Run Anyway.
Download on MacOS

Please note: You will need to allow installs from unknown developers in your security preferences.
Sky Client contains no malicious or harmful code and serves only the purpose of launching our client. If you'd like to view the source code to verify this, please reach out to Lee or Ellis. If you experience any issues - do not hesitate to contact hotel administration in-game or by joining our discord server.
